# victoria3-cli

An economic supply & demand market trade simulator game running in CLI, designed as coding & economics practice. Inspired by Paradox Games like Europa Universalis 5 and Victoria 3.

## Core Mechanics & Features

- **Raw Resource Gathering** - Extract resources from owned regions
- **Buildings** - Process raw resources into refined goods
- **Population System** - Dynamic population with varying product demands
- **Trade Networks** - Multiple markets and intercountry trade
- **Expansion** - Grow your influence through region expansion
- **Economic System** - Money resource management
